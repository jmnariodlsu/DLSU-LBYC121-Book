12 Review: Hessian Matrix and Bordered Hessian

The Hessian is the matrix of second partial derivatives

For: \(f(x,y)\)

12.0.1 Structure of Hessian

\(H = \begin{bmatrix} f_{xx} & f_{xy} \\ f_{yx} & f_{yy} \end{bmatrix}\)

Steps:

  1. Load packages (Deriv)

  2. Define the function

  3. Get the first partial derivatives

  4. Get the second derivatives: \(f_{xx}, f_{yy}, f_{xy}, f_{yx}\)

  5. Check Young’s Theorem

  6. Evaluate at critical point (0,0)

  7. Construct Hessian matrix

  8. Compute the determinant (det(H))

  9. Interpret

12.1 Example in Class

\(C(Q_1, Q_2) = Q_1^2+Q_2^2\)

Use the same steps

12.1.1 General Hessian Template

library(Deriv)

code_hessian <- function(f, x0, y0){

  fx <- Deriv(f,"x")
  fy <- Deriv(f,"y")

  fxx <- Deriv(fx,"x")
  fyy <- Deriv(fy,"y")
  fxy <- Deriv(fx,"y")
  fyx <- Deriv(fy,"x")

  fxx_val <- fxx(x0,y0)
  fyy_val <- fyy(x0,y0)
  fxy_val <- fxy(x0,y0)
  fyx_val <- fyx(x0,y0)

  H <- matrix(c(
    fxx_val, fxy_val,
    fyx_val, fyy_val
  ), nrow=2, byrow=TRUE)

  det_H <- det(H)

  young <- all.equal(fxy_val, fyx_val)

  type <- if(det_H > 0 & fxx_val > 0){
    "Local Minimum"
  } else if(det_H > 0 & fxx_val < 0){
    "Local Maximum"
  } else if(det_H < 0){
    "Saddle Point"
  } else {
    "Inconclusive"
  }

  list(
    Hessian = H,
    Determinant = det_H,
    YoungTheorem = young,
    Type = type
  )
}

This function automates the Hessian procedure

code_hessian<-function (f, x0, y0) creates a function that accepts:

  • f = the original function

  • x0, y0 = the point where curvature will be evaluated.

The function returns:

  • Hessian matrix

  • Determinant

  • Young’s Theorem check

  • classification

12.2 Bordered Hessian

The bordered Hessian is a matrix used when we optimize with a constraint.

Without constraint: Hessian

With constraint: bordered Hessian

The constraint changes the curvature analysis, so we include both:

  • curvature of the Lagrangian

  • derivatives of the constraint

12.2.0.1 Setup

Suppose

Objective function is: \(f(x,y)\)

Constraint is: \(g(x, y)=0\)

Lagrangian: \(\mathcal{L}=f(x,y)+\lambda g(x,y)\)

\(BH = \begin{bmatrix} 0 & g_x & g_y \\ g_x & \mathcal{L}_{xx} & \mathcal{L}_{xy} \\ g_y & \mathcal{L}_{yx} & \mathcal{L}_{yy} \end{bmatrix}\)

12.3 Example in Class:

\(f(x,y)=\sqrt{x}\sqrt{y}\)

Constraint: \(100-10x-20y=0\)

Steps:

  1. Define the Lagrangian equation
  2. Load packages (nleqslv and Deriv)
  3. Define the functions
  4. Set up Lagrangian
  5. Get the first order conditions
  6. Solve for the optimal bundle
  7. Read the solution
  8. Get the second derivatives
  9. Get the constraint derivatives
  10. Build Bordered Hessian
  11. Get determinant
