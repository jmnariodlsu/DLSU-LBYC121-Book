5 Differentiation Rules

5.1 Introduction

Differentiation measures the instantaneous rate of change of a function wrt its variable. We use differentiation in getting the marginal cost, marginal utility or the marginal product.

In R, derivatives are computed symbolically (using expressions) or numerically.

5.2 Structure of Differentiation

D() is R’s built-in symbolic differentiation function. It takes a mathematical expression and returns its derivative wrt a variable.

D(expr, name)

expr is the function you want to differentiate

name is the variable you differentiate with respect to.

Example:

D(expression(x^2), "x")
## 2 * x

There is a need to put expression, because without it, R cannot do it. For example I made x have a value because if it is simply \[x^3\] there will be an error. Here is the difference:

D(expression(x^3), "x")
## 3 * x^2
x<-3
D((x^3), "x")
## [1] 0

Basically, the expression preserves the equation that you are trying to differentiate.

5.3 Constant Rule

If \[f(x)=c\] where c is a constant, then \[f'(x)=0 \]

5.3.1 Example

\[ f(x)=10 \]

Derivative: \[f'(x)=0\]

5.3.1.1 R Implementation

D(expression(10), "x")
## [1] 0

5.4 Power Rule

If \[f(x)=x^n\] then \[f'(x)=nx^{n-1}\]

5.4.1 Example

\[ f(x)=x^3 \]

Derivative: \[f'(x) = 3x^2\]

5.4.1.1 R Implementation

D(expression(x^3),"x")
## 3 * x^2

5.4.2 Example

\[ f(x)=x^{1/2} \]

Derivative:

\[ f'(x)=\frac{1}{2}x^{\frac{-1}{2}} \]

5.4.2.1 R Implementation

D(expression(x^(1/2)), "x")
## x^((1/2) - 1) * (1/2)

5.5 Constant Multiple Rule

If \[f(x)=a*g(x)\] then \[f'(x) = a*g'(x)\]

5.5.1 Example

\[ f(x)=5x^2 \]

Derivative: \[f'(x)=10x\]

5.5.1.1 R Implementation

D(expression(5*x^2), "x")
## 5 * (2 * x)

5.6 Sum and Difference Rule

If \[f(x)=g(x)+h(x)\] then \[f'(x)=g'(x)+h'(x)\]

Same applies for subtraction.

5.6.1 Example

\[ f(x)=x^2+3x+9 \]

Derivative: \[f'(x)=2x+3\]

5.6.1.1 R Implementation

D(expression(x^2+3*x+9), "x")
## 2 * x + 3

5.7 Polynomial Functions (Combined Rules)

Most economic functions are polynomials (i.e. cost function)

5.7.1 Example

\[ C(q)=2q^3-5q^2+10q-100 \]

This is a cost function.

Derivative (Marginal Cost): \[MC(q)=6q^2-10q+10\]

5.7.1.1 R Implementation

D(expression(2*q^3-5*q^2+10*q+100), "q")
## 2 * (3 * q^2) - 5 * (2 * q) + 10

5.8 Exponential Functions

If \[f(x)=e^x\] then \[f'(x)=e^x\]

If \[f(x)=a^x\] then \[f'(x)=a^x ln(a)\]

5.8.1 Example

\[ f(x)=e^x \]

5.8.1.1 R Implementation

D(expression(exp(x)), "x")
## exp(x)

5.8.2 Example

\[ f(x)=2^x \]

D(expression(2^x), "x")
## 2^x * log(2)

5.9 Logarithmic Functions

If \[f(x)=ln(x)\] then \[f'(x)=\frac{1}{x}\]

If \[f(x)=ln(g(x))\] then \[f'(x)=\frac{g'(x)}{g(x)}\]

This is usually for elasticities, utility or diminishing returns

5.9.1 Example

\[ f(x)=ln(x) \]

5.9.1.1 R Implementation

D(expression(log(x)), "x")
## 1/x

5.9.2 Example

\[ f(x)=ln(5x) \]

5.9.2.1 R Implementation

D(expression(log(5*x)), "x")
## 5/(5 * x)

5.10 Product Rule

If \[f(x)=g(x)h(x)\] then \[f'(x)=g'(x)h(x)+g(x)h'(x)\]

This is especially used for Revenue = Price*Quantity

5.10.1 Example

\[ R(q)=q(10-q) \]

Derivative: \[R'(q)=10-2q\]

5.10.1.1 R Implementation

D(expression(q*(10-q)), "q")
## (10 - q) - q

5.11 Quotient Rule

If \[f(x)=\frac{g(x)}{h(x)}\] then \[f'(x)=\frac{g'(x)h(x)-g(x)h'(x)}{h(x)^2}\]

This is used for average cost or average product

5.11.1 Example

\[ AC(q)=\frac{2q^2+10}{q} \]

5.11.1.1 R Implementation

D(expression((2*q^2+10)/q),"q")
## 2 * (2 * q)/q - (2 * q^2 + 10)/q^2

5.11.2 Example

If you are given the following:

\[ TC(q)=q^3+4q^2+10q+100 \]

so, you need to construct average cost:

\[ AC(q)=\frac{q^3+4q^2+10q+100}{q} \]

5.11.2.1 R Implementation

D(expression((q^3+4*q^2+10*q+100)/q),"q")
## (3 * q^2 + 4 * (2 * q) + 10)/q - (q^3 + 4 * q^2 + 10 * q + 100)/q^2

5.12 Chain Rule

If \[f(x)=g(h(x))\] then \[f'(x)=g'(h(x))\cdot h'(x)\]

This is used in CES functions

5.12.1 Example

\[ f(x)=ln(x^2+1) \]

Derivative: \[f'(x)=\frac{2x}{x^2+1}\]

5.12.1.1 R Implementation

D(expression(log(x^2+1)), "x")
## 2 * x/(x^2 + 1)

5.13 Things to Note:

  1. You have to check that you are typing the correct codes, equations, etc.
  2. You need to write expression() completely as exp is different.
  3. Always check the parentheses, commas, etc.
