11.2 Young’s Theorem

Suppose we have a function with two variables:

\(f(x,y)\)

We can take derivatives in different orders:

Differentiate by \(x\), then \(y\)

\(\frac{\partial^2f}{\partial y \partial x}\)

OR

Differentiate by \(y\), then \(x\)

\(\frac{\partial^2f}{\partial x \partial y}\)

Young’s Theorem says that if the function is smooth (continuous derivatives),

\(f_{xy} = f_{yx}\)

Then the order of differentiation does not matter.

In economics, cross partial derivative measures how one variable affects the marginal effect of another.

Example, utility, the cross derivative tells you how consuming more \(y\) changes marginal utility of \(x\).

11.2.1 Utility Function

\(U(x,y) = x^{0.5} y^{0.5}\)

rm(list=ls())
11.2.1.0.1 Define the function
U<- \(x,y) x^0.5*y^0.5
11.2.1.0.2 First partial derivative with respect to X
ux<-Deriv(U,"x")

We computed here the derivative wrt x

11.2.1.0.3 Cross partial derivative (x then y)
uxy<-Deriv(ux,"y")

Takes derivative again, now wrt y

11.2.1.0.4 First partial derivative with respect to Y
uy<-Deriv(U,"y")

We did the same, this time wrt y

11.2.1.0.5 Cross partial derivative (y then x)
uyx<-Deriv(uy,"x")

Take derivative again wrt to x

11.2.1.0.6 Check if identical
identical(uxy, uyx)
## [1] TRUE

We compare the expression. IF TRUE, Young’s theorem holds. If FALSE, Young’s theorem doesn’t.

11.3 Hessian Matrix

The Hessian is the matrix of second partial derivatives.

For:

\(f(x,y)\)

11.3.0.1 Structure of Hessian

To write inline, put what is in the code chunk below.

H = \begin{bmatrix}
f_{xx} & f_{xy} \\ #\\ for new row
f_{yx} & f_{yy}
\end{bmatrix}

\(H = \begin{bmatrix} f_{xx} & f_{xy} \\ f_{yx} & f_{yy} \end{bmatrix}\)

Economically, the Hessian tells the maximum, minimum and saddle point.

Let’s look at the meaning of each part of the matrix:

  • \(f_{xx}\) = curvature in \(x\) direction

  • \(f_{yy}\) = curvature in \(y\) direction

  • \(f_{xy}, f_{yx}\) = cross effects (interaction curvature)

Young’s Theorem is important here because the Hessian becomes symmetric with Young’s Theorem.

11.3.1 Decision Rule (2 variables)

Compute:

\(D=f_{xx}f_{yy}-(f_{xy})^2\)

  • If \(D>0\) and \(f_{xx}>0\), local minimum

    • This is showing cost minimization

  • If \(D>0\) and \(f_{xx}<0\), local maximum

    • This is showing profit or utility maximization

  • If \(D<0\), saddle point

    • shows unstable equilibrium

11.3.2 Local Minimum

\(f(x,y) = x^2+y^2\)

rm(list=ls())

Marginal effects

fxx<-Deriv(fx,"x")

11.3.2.0.8 Interpretation

You can include this, or you can simply look at det_H and fxx_0.

ch<-function(det_H, fxx_0){

if(det_H > 0 & fxx_0 > 0){
  return("Local Minimum")
} else if (det_H > 0 & fxx_0 < 0){
    return("Local Maximum")
} else if (det_H < 0){
  return("Saddle Point")
} else {return("Inconclusive")}
}
  • function(){} creates a reusable function. It checks what is inside the parenthesis then looks at the conditions you set.

  • if(condition){} tells R whether condition is TRUE

  • det_H>0 condition checking

  • & is to check if both conditions are TRUE

  • else if if first condition fails, check next condition

  • else if all conditions fail

  • return reveals what the answer is

ch(det_H, fxx_0)
## [1] "Local Minimum"

11.4 Bordered Hessian

The Bordered Hessian is a matrix used to test whether a constrained optimization problem gives maximum, minimum, or saddle point. It is used when you optimize WITH a constraint.

Without constraint: Hessian

With constraint: bordered Hessian

The constraint changes the curvature analysis and we must include how Lagrange multiplier interacts with variables

11.4.0.1 Setup:

Suppose

Objective function is

\(f(x,y)\)

Constraint is:

\(g(x,y)=0\)

Lagrangian

\(\mathcal{L} = f(x,y)+ \lambda g(x,y)\)

\mathcal{L} = f(x,y)+ \lambda g(x,y)

11.4.0.2 Structure of Bordered Hessian

Two variables:

In the code chunk, you place this in your inline Math

BH = \begin{bmatrix} 
   0 & g_x & g_y \\ 
   g_x & \mathcal{L}_{xx} & \mathcal{L}_{xy} \\ 
   g_y & \mathcal{L}_{yx} & \mathcal{L}_{yy} 
   \end{bmatrix}


\(BH = \begin{bmatrix} 0 & g_x & g_y \\ g_x & \mathcal{L}_{xx} & \mathcal{L}_{xy} \\ g_y & \mathcal{L}_{yx} & \mathcal{L}_{yy} \end{bmatrix}\)

  • Top-left is always zero

  • First row/column are the gradient of constraint;

    • how constraint changes when variables change

  • Lower right block is the second derivatives of Lagrangian

11.4.1 Setup Example:

\(f(x,y)=\sqrt{x}\sqrt{y}\)

Constraint:

\(100-10x-20y=0\)

Lagrangian

\(\mathcal{L}=\sqrt{x}\sqrt{y} + \lambda(100-10x-20y)\)
